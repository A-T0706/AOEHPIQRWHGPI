# Black Box Eatz Website

A custom multi-page restaurant website for **Black Box Eatz**, a Black-owned Caribbean and Southern soul food fusion restaurant concept in Brooklyn, New York.

## Pages
- `index.html` — homepage with hero, featured dishes, and call-to-action
- `menu.html` — organized menu sections
- `gallery.html` — visual food gallery made with CSS blocks
- `story.html` — restaurant story and design purpose
- `contact.html` — contact/location page with form layout

## How to publish on GitHub Pages
1. Create a new GitHub repository.
2. Upload all files and folders from this project.
3. Go to **Settings → Pages**.
4. Under **Branch**, choose `main` and `/root`.
5. Click **Save**.
